# CausalLift

CausalLift is a beginner-friendly causal machine learning project for A/B testing and uplift modeling.

## Features

- Traditional A/B test analysis
- T-Learner uplift modeling
- Qini / uplift curve evaluation
- Segment-level treatment effect analysis
- Modular Python architecture

## Tech Stack

- Python
- Scikit-learn
- Pandas
- Matplotlib
- SciPy

## Goal

Estimate treatment effects and identify high-impact user segments using causal inference techniques.
